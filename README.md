# 📊 Sales Data Analysis Dashboard

An interactive Power BI dashboard designed to analyze sales performance, revenue trends, customer behavior, and product insights using the Sales Data Sample dataset.

![Dashboard Preview](dashboard.png)

---

## 📌 Project Overview

This project focuses on transforming raw sales data into actionable business insights through interactive visualizations and KPI tracking.

The dashboard helps stakeholders understand:

- Total sales performance
- Revenue trends over time
- Top-performing product lines
- Customer purchasing behavior
- Regional sales distribution
- Deal size analysis
- Quarterly and yearly growth patterns

---

## 🎯 Objectives

- Analyze overall sales performance
- Identify top-selling products and product lines
- Track monthly, quarterly, and yearly revenue trends
- Understand customer purchasing patterns
- Generate business insights for decision-making
- Build an interactive and professional Power BI dashboard

## 📂 Dataset Information

Dataset Name:

**sales_data_sample.csv**

### Dataset Features

| Column Name | Description |
|-------------|-------------|
| ORDERNUMBER | Unique order identifier |
| QUANTITYORDERED | Quantity of products ordered |
| PRICEEACH | Price per unit |
| SALES | Total sales amount |
| ORDERDATE | Date of order |
| STATUS | Order status |
| QTR_ID | Quarter number |
| MONTH_ID | Month number |
| YEAR_ID | Year |
| PRODUCTLINE | Product category |
| MSRP | Manufacturer suggested retail price |
| PRODUCTCODE | Product code |
| CUSTOMERNAME | Customer name |
| CITY | Customer city |
| STATE | Customer state |
| COUNTRY | Customer country |
| TERRITORY | Sales territory |
| DEALSIZE | Size of the deal |

## 🔄 Project Workflow

### 1. Data Collection

- Imported the Sales Data Sample CSV dataset into Power BI.

### 2. Data Cleaning

Performed data preprocessing using Power Query:

- Removed missing values
- Checked data types
- Formatted date columns
- Removed unnecessary fields
- Validated numerical data

### 3. Data Transformation

Created additional columns such as:

- Year
- Month
- Quarter
- Revenue Metrics
- Sales Categories

### 4. KPI Creation

Developed key business metrics:

- Total Sales
- Total Orders
- Average Order Value
- Total Customers
- Revenue by Product Line
- Quarterly Sales Performance

### 5. Dashboard Development

Built interactive visualizations including:

- KPI Cards
- Line Charts
- Bar Charts
- Pie Charts
- Slicers
- Filters

### 6. Business Insight Generation

Analyzed:

- Best-performing products
- Customer purchasing trends
- Seasonal sales patterns
- Regional performance
- Deal size contributions

## 📈 Key Performance Indicators (KPIs)

The dashboard tracks important business metrics:

### 💰 Total Revenue

Overall revenue generated from all sales transactions.

### 📦 Total Orders

Total number of customer orders.

### 🛒 Average Order Value

Average amount spent per order.

### 🏆 Top Product Lines

Most profitable product categories.

### 🌍 Regional Performance
